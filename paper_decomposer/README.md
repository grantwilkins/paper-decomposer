# paper_decomposer/ (repo-root shim)

This directory is **not** the real package. The real package lives at [../src/paper_decomposer/](../src/paper_decomposer/). This shim exists only so that

```bash
python -m paper_decomposer path/to/paper.pdf
```

works from the repo root **without** needing an editable install (`uv pip install -e .`).

## Files

| File | Purpose |
|------|---------|
| [`__init__.py`](__init__.py) | Calls `pkgutil.extend_path(__path__, __name__)`, then appends `<repo-root>/src/paper_decomposer` to `__path__`. After that, `paper_decomposer.<anything>` resolves to the corresponding module in `src/paper_decomposer/`. The shim also re-exports `ConfigError`, `get_config`, `load_config` as a convenience. |
| [`__main__.py`](__main__.py) | Imports `main` from `.cli` (resolved via the extended path to `src/paper_decomposer/cli.py`) and runs `raise SystemExit(main())`. |

## How path resolution works

1. Python imports `paper_decomposer` and runs this `__init__.py`.
2. `__init__.py` computes `<this-dir>/../src/paper_decomposer` and appends it to `__path__` if it exists.
3. Any subsequent `import paper_decomposer.X` is resolved by scanning `__path__` in order: this directory first, then the `src` directory. Since only `__init__.py` and `__main__.py` live here, every other submodule (`cli`, `config`, `schema`, `models`, `pdf_parser`, `pipeline`, `extraction.*`) is served from `src/paper_decomposer/`.

## When **not** to edit this directory

Do not add pipeline logic here. The shim's only job is path redirection; any real module belongs under [../src/paper_decomposer/](../src/paper_decomposer/). If you install the project as a distribution (e.g., via `uv pip install -e .` with a `[tool.hatch.build]` / src-layout config), this shim becomes unnecessary but is harmless — the extended path simply finds the installed location as well.

## Note on `__pycache__/`

The compiled `.pyc` cache in this directory is generated by `python -m paper_decomposer` runs. It is not checked into git (see root [.gitignore](../.gitignore)).
